# منصة فتح للتعلّم الذكي 🌟

منصة تعليمية ذكية مجانية تفتح لك أبواب المعرفة بتقنيات الذكاء الاصطناعي المتطورة

> **تعليم للجميع - لا أحد يُترك خلف الركب**

## 🚀 الميزات الرئيسية

### 🤖 الذكاء الاصطناعي المتطور
- **مساعد ذكي تفاعلي**: مدعوم بتقنية Google Gemini مع دعم كامل للغة العربية
- **أدوات تلخيص النصوص**: لخص النصوص الطويلة بذكاء مع إحصائيات مفصلة
- **مساعد الكتابة**: تحسين وتطوير النصوص والمقالات

### 🎓 التعليم التفاعلي
- **دورات تعليمية شاملة**: محتوى متنوع للجميع (الأطفال، الشباب، الأسر، كبار السن)
- **مهارات حياتية عملية**: الطبخ، الحرف، الإدارة المالية، والمهارات التقنية
- **تعلم مخصص**: مسارات تعليمية مخصصة حسب احتياجاتك ومستواك
- **تتبع التقدم**: راقب تقدمك في التعلم مع إحصائيات مفصلة

### 🏛️ نظام الهولاكراسي (الإدارة المفتوحة)
- **إدارة شفافة**: نظام إداري يشرك المجتمع في اتخاذ القرارات
- **توزيع المسؤوليات**: حسب الخبرة والاهتمام بدلاً من التسلسل الهرمي
- **فرص القيادة**: إمكانية قيادة مشاريع تعليمية وتطوير المهارات الإدارية
- **مجتمع متطوعين**: انضم كمتطوع أو استفد من خبرات المتطوعين

### 💻 تجربة مستخدم متميزة
- **واجهة عربية متقدمة**: دعم كامل للغة العربية مع RTL وأيقونات احترافية
- **تصميم متجاوب**: يعمل بسلاسة على جميع الأجهزة
- **وضع مظلم/فاتح**: تبديل سهل بين الأوضاع
- **تسجيل متقدم**: نموذج تسجيل على 5 مراحل مع شرح نظام الهولاكراسي

## 🛠️ التقنيات المستخدمة

### Frontend
- **Framework**: Next.js 15 مع TypeScript
- **Styling**: Tailwind CSS v4 مع دعم RTL كامل
- **Icons**: Lucide React (أيقونات احترافية)
- **UI Components**: مكونات مخصصة مع تصميم متجاوب

### Backend
- **API**: Next.js API Routes
- **ORM**: Prisma مع PostgreSQL
- **Authentication**: NextAuth.js مع دعم Google OAuth
- **File Upload**: دعم رفع الملفات والصور

### الذكاء الاصطناعي
- **AI Provider**: Google Gemini API (بدلاً من OpenAI)
- **Features**: دردشة ذكية، تلخيص النصوص، مساعد الكتابة
- **Language Support**: دعم متقدم للغة العربية

### قاعدة البيانات
- **Database**: PostgreSQL 14+
- **ORM**: Prisma مع TypeScript
- **Migrations**: إدارة تلقائية للمخطط

## 📋 متطلبات التشغيل

- **Node.js**: 18+ (يُفضل 20+)
- **قاعدة البيانات**: PostgreSQL 14+ أو MySQL 8+
- **مفتاح AI**: Google Gemini API (مجاني مع حدود سخية)
- **المتصفح**: أي متصفح حديث مع دعم ES2020+

## ⚡ التثبيت والتشغيل

### 1. تثبيت المكتبات
```bash
npm install
```

### 2. إعداد متغيرات البيئة
انسخ ملف `.env.example` إلى `.env` وأضف القيم المطلوبة:

```bash
cp .env.example .env
```

املأ المتغيرات التالية في ملف `.env`:
```env
# Database (PostgreSQL أو MySQL)
DATABASE_URL="postgresql://username:password@localhost:5432/fateh_platform_db"
# أو للـ MySQL مع XAMPP:
# DATABASE_URL="mysql://root:@localhost:3306/fateh_platform_db"

# NextAuth.js
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="fateh-platform-secret-key-2025-very-secure-random-string"

# Google OAuth (اختياري)
GOOGLE_CLIENT_ID="your-google-client-id"
GOOGLE_CLIENT_SECRET="your-google-client-secret"

# Google Gemini API (مجاني)
GEMINI_API_KEY="your-gemini-api-key"

# App Configuration
NODE_ENV="development"
```

### 🔑 الحصول على مفتاح Gemini API
1. اذهب إلى [Google AI Studio](https://makersuite.google.com/app/apikey)
2. سجل الدخول بحساب Google
3. انقر على "Create API Key"
4. انسخ المفتاح وضعه في متغير `GEMINI_API_KEY`

### 3. إعداد قاعدة البيانات
```bash
# إنشاء قاعدة البيانات وتطبيق المخطط
npm run db:push

# إنشاء البيانات التجريبية
npm run db:seed
```

### 4. تشغيل المشروع
```bash
npm run dev
```

المشروع سيعمل على: http://localhost:3000

- **دعم السياق**: فهم محتوى الدرس والمحادثة السابقة
